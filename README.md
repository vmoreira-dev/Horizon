# Horizon

[![Horizon UI](./public/assets/horizon_5_4.webp)](./public/assets/horizon_5_4.webp)

---

## Overview

Horizon is a minimal task board structured around fixed columns and continuous interaction.


[Live Demo](https://horizon.vmoreira.dev)

---

## Design

- Dark gradient background behind a fixed three-column board
- Cards move instantly between columns
- The board stays visible during interaction




---

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Prisma ORM
- PostgreSQL (NeonDB)
