# Horizon

[![Horizon UI](./public/assets/horizon_5_4.webp)](./public/assets/horizon_5_4.webp)

---

## Overview

Horizon is a minimal task board structured around fixed columns and continuous interaction.


[View](https://horizon.vmoreira.dev)

---

## Design

- Minimal surface area  
- Fast, direct interaction  
- Persistent state  
- Low-friction by default  


---

## Stack

- Next.js 15 (App Router, Turbopack)  
- React 19  
- TypeScript  
- Tailwind CSS (v4)  
- Prisma ORM (v6)  
- PostgreSQL (NeonDB)
